# Sprintura Demo (personal/demo license — Ahweproject)

Bundled web fonts for the Streamlit dashboard:

- `Sprintura-Demo.woff2`
- `Sprintura-Demo.woff`

## Windows: use your local download folder

If you have additional formats from:

`C:\Users\USER\Downloads\sprintura-demo`

Copy any of these into this folder (same names preferred):

- `Sprintura-Demo.otf` / `Sprintura-Demo.ttf`
- `Sprintura Regular.woff2` (rename to `Sprintura-Demo.woff2` if needed)

Restart Streamlit after copying fonts.

The theme loads `./static/fonts/Sprintura-Demo.woff2` relative to `app/streamlit_app.py`.
